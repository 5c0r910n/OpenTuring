Str

Description   This unit contains the predefined constants for manipulating strings.

Several routines in the Str module are exported unqualified.

Entry Points  

Lower Convert a string to lower case.
Upper Convert a string to upper case.
Trim Remove leading and trailing blanks.
Pad Pads a string to a specified length by adding spaces after.
Centre Centres a string in a field of a specified length by adding spaces before and after.
Center Centers a string in a field of a specified length by adding spaces before and after.
index Finds a specified string in another string.
length Returns the length of a string.
repeat Creates a string by repeating a specified string a number of times.
Left Returns the left-most specified number of characters in a string.
Right Returns the right-most specified number of characters in a string.
StartsWith Returns true if a string starts with a specified string.
EndsWith Returns true if a string ends with a specified string.
ReadLineFromCharArray Copies a line of text from a character array into a string.
WriteLineToCharArray Writes a line of text from a string into a character array.
WriteToCharArray Writes text from a string into a character array.
CharArrayCopy Copies a section of a character array into another character array.
